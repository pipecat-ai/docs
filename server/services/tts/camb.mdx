## Overview

`CambTTSService` provides high-quality text-to-speech synthesis using Camb AI's MARS model family with streaming capabilities. The service offers multiple model options optimized for different use cases: `mars-flash` for fast inference and `mars-pro` for high-quality output.

## Installation

To use Camb AI services, install the required dependencies:

```bash
pip install "pipecat-ai[camb]"
```

## Prerequisites

### Camb AI Account Setup

Before using Camb AI TTS services, you need:

1. **Camb AI Account**: Sign up at [Camb AI](https://studio.camb.ai/)
2. **API Key**: Generate an API key from your account dashboard
3. **Voice Selection**: Choose voice IDs from the platform

### Required Environment Variables

- `CAMB_API_KEY`: Your Camb AI API key for authentication

## Configuration

### Constructor Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `api_key` | `str` | Required | Camb AI API key for authentication |
| `voice_id` | `int` | `147320` | Voice ID to use for synthesis |
| `model` | `str` | `"mars-flash"` | TTS model: `"mars-flash"` (fast) or `"mars-pro"` (high quality) |
| `timeout` | `float` | `60.0` | Request timeout in seconds |
| `sample_rate` | `int` | Model default | Audio sample rate in Hz (22050 for mars-flash, 48000 for mars-pro) |
| `params` | `InputParams` | `None` | Additional voice parameters |

### Input Parameters

The `InputParams` class allows configuring language and model-specific options:

```python
class InputParams(BaseModel):
    language: Optional[Language] = Language.EN
    user_instructions: Optional[str] = None  # For mars-instruct model only
```

## Usage Examples

### Basic Usage

```python
from pipecat.services.camb.tts import CambTTSService

# Using mars-flash (fast inference)
tts = CambTTSService(
    api_key=os.getenv("CAMB_API_KEY"),
    model="mars-flash",
)
```

### High Quality Mode

```python
from pipecat.services.camb.tts import CambTTSService

# Using mars-pro for higher quality
tts = CambTTSService(
    api_key=os.getenv("CAMB_API_KEY"),
    model="mars-pro",
    voice_id=12345,
)
```

### With Language Configuration

```python
from pipecat.services.camb.tts import CambTTSService
from pipecat.transcriptions.language import Language

tts = CambTTSService(
    api_key=os.getenv("CAMB_API_KEY"),
    model="mars-flash",
    params=CambTTSService.InputParams(
        language=Language.ES,
    ),
)
```

## Models

| Model | Sample Rate | Description |
|-------|-------------|-------------|
| `mars-flash` | 22.05kHz | Fast inference, optimized for real-time applications |
| `mars-pro` | 48kHz | High-quality output for premium audio |
| `mars-instruct` | 22.05kHz | Supports custom instructions for tone/style control |

## Supported Languages

Camb AI supports 140+ languages. Common language codes include:

- English (US): `en-us`
- English (UK): `en-gb`
- Spanish: `es-es`
- French: `fr-fr`
- German: `de-de`
- Japanese: `ja-jp`
- Chinese (Simplified): `zh-cn`
- And many more...

## Frame Flow

```
TextFrame → CambTTSService → TTSAudioRawFrame
                          → TTSStartedFrame
                          → TTSStoppedFrame
```

## Notes

- The service automatically uses model-specific sample rates if not explicitly configured
- Text is limited to 3000 characters per request
- For real-time applications, `mars-flash` is recommended due to lower latency
- The `mars-instruct` model supports `user_instructions` for controlling tone and style
