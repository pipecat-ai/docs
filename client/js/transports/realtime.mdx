## Overview

The Pipecat client SDKs offer multiple transport packages for real-time communication with a Voice-to-Voice LLM servers. The RealTimeWebSocketTransport provides an abstraction for establishing direct connections to LLM servers and handling media streaming.

<Note>
  Transports of this type are design primarily for development and testing
  purposes. For production applications, you will need to build a server
  component with a server-friendly transport, like the
  [DailyTransport](./daily), to securely handle API keys.
</Note>
